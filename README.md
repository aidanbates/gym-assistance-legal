### Features to Test

2. **Track a Run**: Tap Running tab > Start Run (requires location permission)
3. **View Progress**: Profile > Analytics to see strength charts
4. **Check History**: Profile > Workout History
5. **Settings**: Profile > Settings to change units (kg/lbs, km/miles)
